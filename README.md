# Voice Input for macOS 🎙️

按住 Fn 键说话，松开自动转录并输入文字。

## 使用方法

1. **按住 Fn 键** → 开始录音
2. **说话** → 实时转录
3. **松开 Fn 键** → 自动输入文字

## 安装

```bash
git clone https://github.com/davard123/voice-input-mac.git
cd voice-input-mac
make build
make install
```

## 系统要求

- macOS 14.0+
- 麦克风权限
- 语音识别权限

## 功能

- ✅ 全局 Fn 键监听
- ✅ 实时语音转文字
- ✅ 多语言支持（中文/英文/日文/韩文）
- ✅ 精致悬浮窗 UI
- ✅ 实时音频波形动画
- ✅ 菜单栏常驻
